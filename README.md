## Описание проекта
Проект посвящен анализу различий между классическими IT-проектами и финтех-проектами, а также разработке прототипа системы управления рисками.

## Структура репозитория
- `/analysis` — Сравнительная таблица и теоретический анализ (Часть 1).
- `/src` — Исходный код прототипа на Python с ML-моделью (Часть 2).
- `/docs` — Проектная документация модуля (Часть 3).
